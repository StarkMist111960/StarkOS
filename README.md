
STARKOS

StarkOS was a project I made because I have always had a love for lower level software. This OS is more of a passion project than anything, and i would like to share it with others. I made this myself in order to leanr more about how this stuff works. I used the grub bootloader because asm is incredibly annoying, everything else I made myself.

If this damages your hardware I am not responsible.


Use the following command to build the OS using the including bash file:

```
bash <(curl -Ls https://raw.githubusercontent.com/StarkMist111960/StarkOS/main/build/build.sh) 
```
